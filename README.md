# 音乐源
为防止版权问题，你必须使用自己的音乐索引。你需要编制一个包含封面、歌词、音频三个文件url的JSON。

## 配置文件
下面展示了一个可能的歌曲列表（包含没有源数据的嗵嗵歌曲）
```json
[
  {
    "id": "1",
    "title": "嗵嗵",
    "base":"嗵嗵",
    "artist": "DOUDOU",
    "coverUrl": "",
    "audioUrl": "",
    "lyricUrl": ""
  }
]
```

在填写完音乐源后，你需要将音乐院保存到任何可以直接用网页程序访问的地方，例如一个文件床、openlist....等，然后将地址填入播放器的设置中。

# 主题
为了凸显这是一个DOUDOU的播放器（也许更多的是允许用户创建一个独属于自己的小音乐盒），我为DOUDOU/福禄寿的歌曲添加了一系列独有的色彩。

## 基调
DOUDOU的给了我极大的力量...我认为我也可以用不同的颜色来表现其内核...

### 南三环东路

> 污泥啊，血肉啊，石头和大地也是你啊；莫要哭，莫要怕，走过了东路就到家

南三环东路，2023年6月作为DOUDOU组合的回归首作发布，也是我重新找到组织的一首歌（有趣的是我在洗澡时候听到甚至感觉谁模仿的太像了!

虽然不想提，但是那段艰苦的时光中，我们无数人，都会在一个深夜，也许不是呐喊，只是默默的流泪...

我知道那种朦朦胧胧的灰暗的感觉，我知道那种闪烁路灯下的疲惫，和渐渐暗淡的月色，静默的晚上，无数的我，我们。

南三环东路这首歌，我觉得他是一种蓝色，一种质朴、平和、却又被生活蒙上了灰色的颜色，一种被迷雾包裹的灰蓝色。

但我们终会穿透这层蓝色的迷雾，我们终会回到温暖的家。

### 嗵嗵

> 可以啦，可以啦，可以变回孩子啦~

正如豆咪说的，这首歌，讲述了自由自在的生命，在时间的轮回中自由地起舞的故事。
是啊，我们总想回到那片丛林，回到弥漫着绿意的灌木丛，回到清澈的湖畔。在蔚蓝的天空下，大地开满鲜花，夜空高悬繁星，我们围着篝火，欢快地跳舞。

但也许我们都会遭遇一些困难...自由的我们，或许会被关进无形的囚笼，或许会被不知名的妖魔鬼怪挡住去路。

不过没关系！毛茸茸的小家伙们会继续起舞的，他们会围着这颗星球继续旅行，转着圈，蹦着跳，哼着轻快的歌。当树上的果实落下时，我们就用最快乐最期待的感受去迎接那一声：

**嗵！**

写完上面的几句话突然好开心，我感觉自己也变成了一个快乐的毛毛球...那我一定是绿色的吧，嘿嘿。

我让嗵嗵整体呈现一种「稍微有一点点深的绿色」，一种泥土+树叶的自然气息，一种浑厚而原始的生命力！
