# 音乐源
为防止版权问题，你必须使用自己的音乐索引。你需要编制一个包含封面、歌词、音频三个文件url的JSON。

## 配置文件
下面展示了一个可能的歌曲列表（包含没有源数据的嗵嗵歌曲）
```json
[
  {
    "id": "1",
    "title": "嗵嗵",
    "base":"嗵嗵",
    "artist": "DOUDOU",
    "coverUrl": "",
    "audioUrl": "",
    "lyricUrl": ""
  }
]
```

在填写完音乐源后，你需要将音乐院保存到任何可以直接用网页程序访问的地方，例如一个文件床、openlist....等，然后将地址填入播放器的设置中。

### 南三环东路

> 污泥啊，血肉啊，石头和大地也是你啊；莫要哭，莫要怕，走过了东路就到家

南三环东路，2023年6月作为DOUDOU组合的回归首作发布，也是我重新找到组织的一首歌（有趣的是我在洗澡时候听到甚至感觉谁模仿的太像了!

虽然不想提，但是那段艰苦的时光中，我们无数人，都会在一个深夜，也许不是呐喊，只是默默的流泪...

我知道那种朦朦胧胧的灰暗的感觉，我知道那种闪烁路灯下的疲惫，和渐渐暗淡的月色，静默的晚上，无数的我，我们。

但我们终会穿透这层蓝色的迷雾，我们终会回到温暖的家。
